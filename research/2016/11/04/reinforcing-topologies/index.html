Designing Neural Network Architectures Using Reinforcement Learning

04 Nov 2016

We model Convolutional Neural Network (CNN) architecture selection as a Markov Decision Process. The system can be distributed across many machines and generates models that out perform all others that use the same layer types. Caffe code for the top models generated can be found on my github, and the paper can be found on arxiv or the openreview site for ICLR 2017.
